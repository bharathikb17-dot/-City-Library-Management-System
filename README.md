\# 📚 City Library Management System



\## 📌 Description

A Python-based Library Management System with:

\- Book \& Member Management

\- Borrow \& Return System

\- Due Dates \& Fine Calculation

\- Data Storage using CSV

\- Analytics Dashboard



\---



\## 🚀 Technologies

\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Streamlit



\---



\## ▶️ How to Run



```bash

pip install -r requirements.txt

streamlit run app.py
